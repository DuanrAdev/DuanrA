## 📌 About Me
Hey there! I'm **Arnaud** — a freshly converted (and partly corrupted) web developer.  
I traded my previous life for a world where semicolons ruin evenings, APIs ghost you like bad Tinder matches, and deploying to production is an extreme sport.

After investing a *questionable* amount of time and money, I now build things that sometimes work, sometimes explode, and occasionally do both at the same time.  
My code is mostly readable, my curiosity is endless, and my imposter syndrome only screams on weekdays.

If you're here, welcome to the chaos. 🚀
